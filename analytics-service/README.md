## Overview

TerraQuake is a comprehensive seismic data platform that combines a powerful REST API built with Node.js and a dedicated analytics microservice developed with Django and Django REST Framework. It provides researchers, developers, and organizations with easy access to real-time earthquake data, advanced filtering capabilities, and intelligent geospatial analysis. Through its modular architecture, TerraQuake enables not only data visualization and querying, but also statistical processing and machine learning-based insights on seismic activity.
